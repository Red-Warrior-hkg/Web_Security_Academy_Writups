## 📖 About This Repository

This repo contains my **personal writeups** for every lab on [PortSwigger Web Security Academy](https://portswigger.net/web-security) — one of the best free resources for learning real-world web application security.

Each writeup covers:
- 🔍 **Vulnerability explanation** — what it is and why it happens
- 🧪 **Lab analysis** — what the target is and what tools are needed
- 🛠️ **Step-by-step exploitation** — exactly how I solved it
- 💡 **Key takeaways** — the lesson behind the lab

> Labs are solved using **Burp Suite**, **manual testing**, and custom scripts where needed.
