## 📖 Overview

This report documents the hands-on exploitation of SQL injection vulnerabilities across 5 progressive labs on the **PortSwigger Web Security Academy** platform. Each lab targets a specific SQL injection technique, building from basic WHERE clause manipulation to full credential extraction via UNION attacks.
